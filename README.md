# 💂‍♂️ Life in the UK Quiz 💂‍♀️

Welcome to the **Life in the UK Quiz**! This is a fun and interactive quiz app designed to test your knowledge about the UK. 🇬🇧

---

## 🚀 Features
- 🎴 **Flashcards**: Flip cards to reveal answers.
- ✅ **Track Your Score**: See how many questions you've answered correctly.
- 🎉 **Confetti Celebration**: Celebrate a perfect score with confetti!
- 🔀 **Shuffle Flashcards**: Refresh the quiz with new random questions.
- 📊 **Progress Bar**: Track your progress as you answer questions.
